# gfx-gtk

Simple bridge library to render into a gtk-rs `GLArea` using the Gfx (pre-ll) library.

It has very limited functionality and lots of limitations - but works for simple cases.

```
[dependencies]
gfx_gtk = "0.1"
```

see `examples/setup.rs`
